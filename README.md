# RecurCIPY

Pythonic CI on top of GitHub Actions

## How to begin:

```sh
# 1. install recurCIPY (TODO: python module is not yet there)
pip install recurcipy

# 2. checkout new branch
git checkout -b my_yaml_ci_written_in_python

# 3. Use hello_world example to create your CI configuration
python -m recurcipy --generate-from-example hello_world
# Following files must be created/updated:
#   ./ci/configs/github_artifact.py - CI configuration written in python
#   ./.github/workflows/*.yaml - generated yaml pipelines 

# 4. Commit and Push updates to remote:
git commit -m "Hello World"
git push --set-upstream origin my_yaml_ci_written_in_python

# 5. Create PR for the pushed branch

# 6. Enjoy Your Hello World CI generated by recurCIPY
```

## How to continue:

#### Option 1:
```git
# 1: Update CI configuration: ./ci/configs/hello_world.py
 
# 2. Generate yaml pipelines:
python -m recurcipy --generate

# 3. commit, push, create PR

# 4. see updates reflected in your CI run
```

#### Option 2:
```git
# 1: Generate CI pipeline from more advanced example, for instance "recurcipy" - which is CI for this library:
python -m recurcipy --generate-from-example recurcipy

# 3. commit, push, create PR

# 4. see updates reflected in your CI run
```

#### Option 3:
```git
# 1: Write you own CI config located in: ./ci/config/<your_config_name>.py
 
# 2. Generate yaml pipelines:
python -m recurcipy --generate

# 3. commit, push, create PR

# 4. see updates reflected in your CI run
```
